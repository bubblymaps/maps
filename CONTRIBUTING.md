# Contributing to Bubbly Maps 💧

Thank you for considering contributing to Bubbly Maps! Your help keeps this project growing and improving.

## How to Contribute

### 1. **Fork the repository**  

Click the "Fork" button at the top-right of the repo page.

### 2. **Clone your fork**  

```bash
git clone https://github.com/YOUR_USERNAME/maps.git
cd maps
```

### 3. Create a new branch

```bash
git checkout -b feature/your-feature-name
```

### 4. Make your changes

- Follow the existing code style

- Add meaningful commits

- Test your changes locally

### 5. Push your branch

```bash
git push origin feature/your-feature-name
```

### 6. Open a PR

- Provide a clear description of your changes

- Link any relevant issues

- Wait for review and feedback

### Guidelines

- Keep commits small and focused

- Write descriptive commit messages

- Update documentation if your changes affect functionality

- Respect [code of conduct](CODE_OF_CONDUCT.md) and be courteous

Please do not make big changes without discussing them first with the maintainers.
