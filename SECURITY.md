# Our security policies

## Reporting a Vulnerability

If you discover a security issue in Bubbly Maps, please **do not open a public issue**.  
Instead, contact us privately:

- [Email us](mailto://support@bubblymaps.org)

Include:  
- A description of the vulnerability  
- Steps to reproduce  
- Any potential impact  

We will respond promptly and work with you to resolve the issue.

## Supported Versions

Only the latest release is actively maintained for security fixes.  
Older versions may not receive updates.
